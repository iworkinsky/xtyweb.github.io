            类的定义

            # 1. 类的声明形式

            class ClassName{
	public:
		公有成员(外部接口)
	protected:
		保护型成员
	private:
		私有成员
};

            “public”（公共部分）：不受限制的访问和调用 “private”（私有部分）：只有在类的内部的成员，或者友元才可以访问 “protect”（保护部分）：类似“private”，只有类的内部成员以及友元或子类才可以调用 PS：1.友元即与原有的类是并列关系的函数，子类则是继承了原有类的新类。（这里不做深究） 2.友元的权限比子类高（你的兄弟比你的儿子重要）

            1. public, protected, private被称为段约束符，可多次使用，未说明的成员都默认为private
            2. 成员可包括数据成员和函数成员
            3. 声明的大括号最后要接分号；
            4. 一个成员只能有一种访问属性 例子
            class Clock{
	int hour, minute, second;
	void SetTime(int h, int m, int s);//为何不用全称
	void ShowTime();
};
h,*h?
	void Clock::Showtime(), Clock::ShowTime?

            [注]: 类可以先声明，再定义，但是，在未给出完整定义之前，不可将类实例化

            # 2. 类的实现

            实现一个类，就是实现类中的每一个成员函数 成员函数的定义有两种方式：类内定义和类外定义 类内：和一般定义方式相同 类外：函数返回类型+类名+ :: +函数名+函数定义

            1. 函数成员的实现可以在类内实现，也可以只在类内声明函数原型，在类外定义
            2. 在类内实现成员函数，函数按照内联函数处理；在类外实现函数时，需要用类的作用域分辨符::
            3. 类内实现函数时，注意不能既出现函数定义，又出现函数原型
            4. 对于成员函数的实现，要注意其不遵循传统的C的函数的工作原理，因为成员函数是可以直接访问和修改本类中的数据成员的，而非在一块单独的内存中自顾自得实现功能
