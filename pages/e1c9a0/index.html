    引用

    # 1. 基本定义

    某变量的引用相当于给该变量起了个别名 引用与被引用变量共享存储单元

    # 2. 初始化方式(定义时必须初始化)

    int &pm = a;右端为变量

int &pm = 2;右端为常量

int x = 1;
const int &px = x;const量引用
注意，对原变量的引用只能缩小不能放大，如int可引用为const int，const int不可引用为int
px = 2;此处不对，因为常量不可改变

    # 3. 引用传参

    函数中传递引用可以直接改变原来的值

    int func(int &pa){
	a++;
	return 0;
}

int a = 1;
func(a);
cout << a << endl;显示2
对于引用在函数传参时可以理解为：引用只是个“名”，不是“值”，所以在栈内不会单独复制一个值出来

    # 4. 引用函数

    • 函数返回引用，实际上返回的是一个存储单元（变量），即“左值”。
    • 因为返回的引用是一个存储单元，所以函数返回后这个单元的生命期应该不会结束，否则返回值将没有意义
    int &func(int *pint){
	return *pint;
}

int main(){
	int a = 10, b;
	b = func(&a)*5;区分&表示取地址还是引用
	func(&a) = 88;
	cout << b << " " << a << endl;输出
}
*a 前有类型：a为指向该类型的指针
   前无类型：*a代指a的指向值
&a 前由类型：a为某变量的引用
   前无类型：&a为a的地址值
