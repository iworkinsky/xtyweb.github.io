开始安装git

# 一，进入下载链接

这个就需要去 Git 官网下载对应系统的软件了，

笔记

官网下载链接https://git-scm.com/ (opens new window)
阿里云镜像链接https://gitforwindows.org/ (opens new window)

我下载的版本是Git-2.40.0-64-bit.exe，接下来我们就对这个版本进行安装工作。

提示

我的建议是全选

# 二，打开Git Bash

在桌面空白处鼠标右键,点击 Git Bash Here

# 三，测试

输入

git -v

出现版本号2.40.0就代表成功了
