    Vuepress上手

    # 1.1 简介

    vuepress 是 Vuejs 官方提供的一个是Vue驱动的静态网站生成器，基于Markdown语法生成网页。简单的说它就是一个快速建设文档站点的工具，在简单配置好功能后，需要做的事情就剩下写好一个个 Markdown 文档，并且可以将其发布到github (opens new window)。

    它可以用语法更加简洁的Markdown语法代替较为复杂的html+css语法，更快速便捷地搭建网页

    以下说明将详细书写最开始的下载安装部分

    官方说明网站 (opens new window)

    # 1.2 下载及简单实用

    前提条件：

    1. 官方要求Node.js >= 8.6，不过经实测18.15.0对vuepress的支持可能出现问题，建议下载16.20.0

    2. 安装包管理工具npm或yarn(以下主要用用npm进行演示)

    下载及使用步骤：

    注意：

    以下的命令均在命令行中进行

    1. 创建并进入项目目录
    mkdir test && cd test

    命令意为在命令行所在目录下创建test文件夹并进入该文件夹

    此外如果想在指定位置打开命令行可以在资源管理器浏览对应文件夹时右键，选择“在终端中打开”，或在地址栏中直接输入cmd

    1. 用包管理工具初始化项目
    yarn init 
npm init

    执行该命令后，命令行会让你先从包名开始，输入包括版本、描述等多个项目的说明，如下

    实际操作时，如果不想一个个对对应说明进行详细设置，可以直接回车让对应描述按照默认值设置，所有说明均设置完后会在项目文件(本例中为test)生成内容如下的package.json文件

    此外，如果想将每项设置均为默认值，可以在最开始初始化的时候如下操作

    npm init -y
yarn init -y
    1. 将 VuePress 安装为本地依赖
    yarn add -D vuepress 
npm install -D vuepress

    npm换源后下载会更快 -D 是指仅将vuepress安装到所在文件夹中，不建议使用-g进行全局安装

    1. 创建README文档
    mkdir docs && echo '# Hello VuePress' > docs/README.md

    该命令是指在test文件夹下创建docs文件夹，并在docs文件夹中创建一个README.md文档，其内容为'# Hello VuePress'

    1. 在package.json中添加调试说明

    打开test文件夹下的package.json文件夹，并在"scripts"中添加

    "docs:dev": "vuepress dev docs",
"docs:build": "vuepress build docs"

    注意，不要忘记在test所在行后面添加逗号！

    1. 在本地启动服务器
    yarn docs:dev 
npm run docs:dev
